# Revenue Manager & Analytics 📊

A professional desktop application designed for tourism SMEs to manage inventory, analyze pricing strategy, and visualize revenue growth.

## 🚀 Features
- **Dynamic Revenue Tracking:** Manage products with an automated SQLite database.
- **Economic Intelligence:** Built-in Price Elasticity of Demand (PED) calculator.
- **Live Analytics:** Real-time data visualization using Matplotlib.
- **Export Capabilities:** Generate CSV financial reports with one click.

## 🛠️ Tech Stack
- **Language:** Python
- **GUI:** PySide6 (Qt)
- **Database:** SQLite3
- **Visualization:** Matplotlib
